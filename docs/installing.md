# JupyterHub Install

## 1. install Packages tool

You need

- npm (Node.js)
- pip (Python)

```
apt install npm pip
```

## 2-1. Install packages with pip

```
pip install jupyterhub jupyterlab
```

## 2-2. Install packages with npm

Maybe you need global install (`-g` option)

```
npm install -g configurable-http-proxy
```
